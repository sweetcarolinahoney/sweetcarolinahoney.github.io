Cube Recipes

Listed below are the current recipes for the Horadric Cube:

  • 3 small rejuvenation potions = 1 full rejuvenation potion
  • 3 gems of the same type and grade (lower than perfect) = 1 higher grade gem of the same type
  • 3 runes of the same type (El - Lem) = 1 higher level rune
  • 2 runes of the same type (Pul - Cham) = 1 higher level rune
  • 1 rune + Key = 1 lower level rune
  • Blacksmith's Whetstone + non-ethereal weapon/armor = fully repaired and recharged weapon/armor
  • Blacksmith's Whetstone + arrows/bolts = fully recharged arrows/bolts
  • Sharpened Chisel + any socketed item = remove socketables from sockets (socketables will be destroyed)
  • Forged Tool + non-socketed normal/superior item = socketed item of the same type (5% chance an item become ethereal)
  • Lesser Malus + base magic/rare/set/unique/crafted/tempered item = exceptional version of the item
  • Greater Malus + exceptional magic/rare/set/unique/crafted/tempered item = elite version of the item
  • Magic Dice + magic item = rerolls stats on the magic item
  • Rare Token + rare item = rerolls stats on the rare item
  • Mystic Orb + magic item + perfect gem = crafted item
  • Kurast Sphere + unique/set item = rerolls stats on the unique/set item
  • Enchanted Malus + normal/superior item = magic or rare item of the same type (2/3 chance for magic, 1/3 for rare)
  • Dye + any weapon/torso/shield/helm = recolored item with the color of dye
  • Demon Shard + magic/rare/set/unique/crafted/tempered item = corrupted item
  • Tempering Catalyst + socketed normal/superior item = tempered item
  • Gem Pile = 3 gems of any grade and type
  • Unidentified Ring = magic, rare, set or unique ring (1/4 chance each)
  • Unidentified Amulet = magic, rare, set or unique amulet (1/4 chance each)
  • Unidentified Rune = 3 any runes
  • Staff of Kings + Viper amulet = Horadric Staff
  • Khalim Flail + Khalim Heart + Khalim Eye + Khalim Brain = Super Khalim Flail
  • Twisted Essence of Suffering + Charged Essence of Hatred + Burning Essence of Terror + Festering Essence of Destruction = Diablo's Soulstone
  • Key of Destruction + Key of Hate + Key of Terror = portal to the Matron's Den/Forgotten Sands/Furnace of Pain
  • Baal's Eye + Diablo's Horn + Mephisto's Brain = portal to Uber Tristram
  • Wirt's leg = portal to the Secret Cow Level
  • Cursed Skull = portal to the Skeleton Passage
  • The Secret Seal of Solomon = portal to Goetia
  • Goetia Seal + magic jewel + Mystic Orb = crafted jewel

Stacking

Cube an item to turn it into a stackable version of the item. Cube stackable version of an item with a single item to add to the stack. Cube stackable version of the item by itself to get a single item off the stack. Maximum quantity of the stack is 10.

List of items that can be stacked:

  • Perfect and flawless gems
  • Runes
  • Blacksmith's Whetstone
  • Sharpened Chisel
  • Forged Tool
  • Lesser Malus
  • Greater Malus
  • Magic Dice
  • Rare Token
  • Mystic Orb
  • Kurast Sphere
  • Enchanted Malus
  • Demon Shard
  • Tempering Catalyst
  • Dyes
